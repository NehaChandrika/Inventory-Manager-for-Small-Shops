# Inventory Management System
A simple command-line based inventory managment system built in Python.
Problem it solves: Helps shop owners track stock and sales.

##  Features
Add/remove items.
Update quantity.
Generate low-stock alerts.
Edge case: Item not found, negative quantity, duplicate entries.

##  Learning Outcome
- Strengthened understanding of dictionaries, loops, conditionals, and user input validation

##  How to Run
1. Make sure Python is installed
2. Run the script:
   ```bash
     python parking_system.py
   ```
